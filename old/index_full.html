About Me

Hi again! My name is Jonathan and I am in the Columbus, Ohio area. I have 4+ years of experience as a Front End Web Developer. I have primarily built responsive, interactive, and media rich web apps, but I've created games from time to time as well.

I started learning programming with C and C++. Then moved over to scripting languages, and then web development languages. What I learned originally has carried over to the tools I use now and the new tools that I learn.

My primary toolset consists of HTML5, CSS3, and Javascript with some Actionscript 3 (if anyone still cares). I also have experience with PHP, Sass, Bootstrap, Git, Grunt, Photoshop, Flash, and Maya.

I enjoy taking on coding challenges and creating responsive web interactions.
